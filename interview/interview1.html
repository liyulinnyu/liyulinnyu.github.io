3. The different between "href" and "src" "href": A link between the location of network and the establishment of element(anchor) or document(link). 9. What is a pseudo array in Javascript? How do you transform a pseudo array into a standard array? Pseudo array (class array): cannot call the array method directly or expect any special behavior of the length property, but they can still use the real array traversal method. Typically, the "arguments" of functions, as well as calls to getElementsByTagName, document.childNodes, and so forth, all return NodeList objects that belong to pseudo arrays. You can use Array.prototype.slice.call (fakeArray) to transform an array into a real Array object.
